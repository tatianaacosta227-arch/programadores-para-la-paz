BACKLOG PRIORIZADO

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

2. Como coordinador quiero recibir una notificación en Telegram cuando se registre una nueva incidencia para darle seguimiento oportuno.

3. Como miembro del equipo quiero consultar los reportes registrados para conocer el estado de cada caso.

4. Como administrador quiero actualizar el estado de una incidencia para informar los avances.

5. Como ciudadano quiero consultar las causas disponibles para conocer las iniciativas de la comunidad.
