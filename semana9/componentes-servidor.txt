Sistema operativo:
Es el software principal del equipo o servidor. Administra los recursos del sistema y permite que las aplicaciones funcionen correctamente.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear y administrar servidores web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que recibirán los usuarios cuando accedan al servicio.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad y los eventos del sistema. Ayudan a identificar errores y monitorear el funcionamiento del servidor.
